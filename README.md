# gitc

格式化 git commit message 的小工具

## 为什么要格式化 git commit message

- 提供更多的历史信息 方便快速浏览
- 可以过滤某些 commit (比如文档改动), 便于快速查找
- 可以直接从 commit 生成 change log

header 参考格式: `<type>(<scope>): <subject>`

## 如何使用

- `Ctrl + C` 复制内容
- `Ctrl + R` 重置内容
- `Tab` 快速切换输入框
